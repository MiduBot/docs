Midubot is a bot created by the community to manage and control the Discord server.

1. Install [pnpm](https://pnpm.js.org/):

```md
- Run: npm install -g pnpm
- Check: pnpm -v
```

2. Install dependencies:

```bash
pnpm install
```

3. Create .env file in the root directory:

```bash
DISCORD_TOKEN=
DISCORD_PREFIX=

DISCORD_ID=
DISCORD_PUBLIC=

DISCORD_INTENTS=
```

or copy [.env.example](https://github.com/MiduBot/bot/blob/develop/.env.example):

```bash
cp .env.example .env
```

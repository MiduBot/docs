## 📦 Packages:

- ⚡️ [Nodejs](https://nodejs.org/en/) - JavaScript runtime.
- 🤖 [Discord.js](https://discord.js.org/) - Discord API.
- 📘 [Typescript](https://www.typescriptlang.org/) - TypeScript language.

## 🚀 Getting Started:

1. Install [pnpm](https://pnpm.js.org/):

```bash
1. Run: npm install -g pnpm
2. Check: pnpm -v
```

2. Install dependencies:

```
pnpm install
```

3. Create .env file in the root directory:

```
DISCORD_TOKEN=
DISCORD_PREFIX=

DISCORD_ID=
DISCORD_PUBLIC=

DISCORD_INTENTS=
```

or copy [.env.example](https://github.com/MiduBot/bot/blob/develop/.env.example):

```
cp .env.example .env
```
